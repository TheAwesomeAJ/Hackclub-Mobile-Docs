We’re thrilled you want to help improve our documentation! Follow the steps below to get started.

<Note>
  To contribute, you’ll need a GitHub account. If you don’t have one, you’ll need to create it before you can make edits.
</Note>

---

## 1. Editing a Single Page

Found a typo, outdated info, or have something to add? It’s easy!

1. Scroll to the bottom of the page.
2. Click **Suggest an Edit**.
3. This will redirect you to the corresponding .mdx file on GitHub. Make your changes and submit a pull request—your contributions will help everyone!

## 2. Adding a Page, Page Group, Menu, Tab, etc

If you aren't sure what these terms mean, click [here](https://www.mintlify.com/docs/organize/navigation) for the Official Mintlify Documentation on each of these, and more.
